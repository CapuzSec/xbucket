# xbucket  - Enumeration of s3 files if vulnerable
 
#### Requiments

install aws cli https://aws.amazon.com/pt/cli/

#### Install

```
https://github.com/CapuzSec/xbucket.git
cd xbucket
chmod +x xbucket.sh
``` 

#### Exemple of use:

```
./xbucket.sh --bucket/-b bucket-name\n" 
./xbucket.sh --url/-u bucketname.com.br\n" 
```
